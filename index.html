Jawaban No.1 : Website adalah kumpulan halaman web yang dapat diakses melalui internet. Ketika seseorang mengakses sebuah website, permintaan tersebut dikirim ke web server yang menyimpan konten dan data website tersebut. Web server bertindak sebagai penyedia layanan, mengirimkan halaman web yang diminta ke perangkat pengguna. Domain, seperti www.example.com, adalah alamat yang digunakan untuk mengidentifikasi alamat IP unik web server. Hosting merujuk pada layanan penyimpanan dan pemuatan konten website di server agar dapat diakses secara online. Jadi, web server menyimpan dan mengirimkan konten, domain memberikan alamat unik, dan hosting menyediakan tempat penyimpanan dan akses ke website. Semua elemen ini saling terkait untuk membuat website dapat diakses oleh pengguna di seluruh dunia.
